## 🚀 Getting started

1. Clone repository: 
```sh
git clone https://github.com/Safi1012/filipesantoscorrea.com
```

2. Install dependencies: 
```sh
npm install
```

3. Start develop server: 
```sh
npm run develop
```

4. Build production: 
```sh
npm run build
```
